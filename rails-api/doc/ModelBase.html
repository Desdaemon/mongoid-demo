Overview

typed: strict

Re-exports several methods that were not provided when including Mongoid::Documentfor Solargraph and other code-completion tools. Note that you still need to manually include Mongoid::Document and other modules.

Examples:

Extend this module in models via the @!parse directive:

class Article
  # @!parse extend ModelBase
  include Mongoid::Document
end
